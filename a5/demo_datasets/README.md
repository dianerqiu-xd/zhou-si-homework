# A5 Demo Datasets

Small image subsets committed with the Streamlit app for cloud demos.
The full local datasets live in `数据集/` and are ignored by Git.

Contents:

- `hog_bow_dataset`: 16 images
  - train: 3 images per class
  - test: 1 images per class
- `cnn_lenet_mnist`: 20 images
  - train: 1 images per class
  - test: 1 images per class
- `resnet_cifar10_dataset`: 10 images
  - train: 0 images per class
  - test: 1 images per class
